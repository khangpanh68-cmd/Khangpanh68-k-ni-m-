## 🚀 Hướng dẫn triển khai (Quick Start)

### 1. Cấu hình nhanh
1.  **Fork** dự án này về tài khoản của bạn.
2.  Mở file `index.html` và tìm các biến sau để chỉnh sửa:
    * `LOVE_DATE`: Thay bằng ngày hai bạn bắt đầu yêu nhau.
    * `PASS`: Thay bằng mật khẩu bạn muốn (Mặc định: `21012023`).
    * `milestones`: Cập nhật các mốc kỷ niệm trong hàm `showTimeline()`.

### 2. Triển khai lên GitHub Pages
Dự án đã tích hợp sẵn **GitHub Actions** (`deploy.yml`). Để kích hoạt:
1.  Vào **Settings** > **Pages**.
2.  Tại mục **Build and deployment > Source**, chọn **GitHub Actions**.
3.  Mỗi khi bạn `push` code mới, trang web sẽ tự động cập nhật tại: `https://<your-username>.github.io/<your-repo-name>/`
